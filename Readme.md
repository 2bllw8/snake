# Snake

```
Score: 11
+------------------------------+
|                              |
|                              |
|                              |
|                              |
|                              |
|         #                    |
|         #                    |
|      ####                    |
|      #                 *     |
|      ##########@             |
+------------------------------+
```

Simple snake game implemented in C.

CLI gui is implemented with ncurses.

## Build

```bash
bazel build //main:snake
```

## Run

```bash
./bazel-bin/main/snake
```
